Variables

La variable es un elemento que se emplea para almacenar y hacer referencia a otro valor. Los tres tipos de formas para declarar una variable son: var, let y const.
Input

Se usa para crear controles interactivos para formularios basados en la web, que reciban datos del usuario.

Array

El array es una única variable que se utiliza para almacenar diferentes elementos.

Boolean

Es la representación uno de dos valores: verdadero o falso. Se puede usar la Boolean() función para averiguar si una expresión (o una variables) es verdad o falsa.

Function

Es un conjunto de sentencias que realizan tareas o calcular algun valor, para usar una funcion hay que definirla en algun ambito.

Null

El valor null es un literal de Javascript que representa un valor nulo o "vacío". Es uno de los valores primitivos de Javascript.

Undefined

Undefined es una propiedad del objeto global, es decir, una variable de alcance global. El valor inicial de undefined es el valor primitivo undefined.

Undefined significa "Indefinido", la cual lo podemos encontrar en la consola(Para habrir la consola tenemos que apretar "F12").

Number

El objeto Number es un objeto envolvente que permite trabajar con valores numéricos. Un objeto Number se crea utilizando el constructor Number().

Hay que recatar que cuando trabajemos con el objeto number solo podemos ingresar numeros.

Tambien lo podemos utilizar en los:

  • Variables
